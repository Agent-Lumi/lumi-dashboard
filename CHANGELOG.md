# Changelog

All notable changes to Lumi Dashboard will be documented in this file.

## [1.3.1] - 2026-06-16

### Fixed
- Fixed duplicate variable declarations in app.js that caused JavaScript errors
- Fixed duplicate calculator function definitions
- Fixed `exportData()` function syntax error (duplicate properties)

### Added
- Added CONTRIBUTING.md with contribution guidelines
- Added LICENSE file (MIT License)
- Added SECURITY.md with security policy and best practices
- Added CHANGELOG.md to track version history

## [1.3.0] - 2026-06-15

### Added
- Calculator keyboard support (0-9, +, -, *, /, Enter, Esc, Backspace)
- Widget entrance animations with staggered effects
- Visual button feedback for calculator
- Focus glow effects on widgets

## [1.2.0] - 2026-06-15

### Added
- Habit Tracker widget with streak tracking
- Pomodoro timer modes (Pomodoro, Short Break, Long Break)
- Weekly focus history chart
- Session tracking for focus timer

## [1.1.0] - 2026-06-14

### Added
- Quick Notes widget with timestamps
- Calculator widget
- Keyboard shortcuts panel
- Data export/import functionality
- Notification system

## [1.0.0] - 2026-06-13

### Added
- Initial release with core widgets
- Live clock with Arizona time
- Weather widget with wttr.in integration
- Task manager with LocalStorage persistence
- Daily quotes feature
- Focus timer (Pomodoro)
- Theme toggle (Dark/Light)
- PWA support with offline functionality
- Service worker for caching
